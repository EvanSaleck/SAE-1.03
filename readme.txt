
Afin de faciliter vos créations de rapports trimestriels, nous avons mis en place un outil et plus précisément un script qui 
permet de récupérer les données de toutes les régions de Frances pour en créer un PDF pour chaque régions.

Avant de se servir de cet outil, vous téléchargerez notre archive en Zip puis vous l'extrayerez, il vous suffira par la suite de placer vos fichiers 
textes comme ceux que vous nous avez fournis dans le dossier 'fichiers.txt'.

Une fois cela fait, il vous faudra ouvrir un terminal sous Linux et utiliser la première  commande : "chmod +x run.sh ./run.sh" 
et pour finir la seconde : "./run.sh", ce qui fera exécuter les instructions permettant la générations des PDF.
